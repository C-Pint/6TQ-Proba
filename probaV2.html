04

Synthèse Théorique

Le vocabulaire et les outils mathématiques essentiels.

A L'expérience aléatoire

Définition

Expérience répétable dont on connaît tous les résultats possibles mais dont le résultat final est dû au seul hasard.

Univers et Issues

  • Issue : Un résultat possible (ex: "Obtenir 6").
  • Univers ($\Omega$) : L'ensemble de toutes les issues.
$\Omega = \{1; 2; 3; 4; 5; 6\}$

B Lexique des événements

Événement

Un sous-ensemble de $\Omega$.
Ex: $A = \{2; 4; 6\}$ (pairs)

Événement élémentaire

Contient une seule issue.
Ex: $E = \{4\}$

Événement impossible

Ne contient aucune issue.
Noté : $\emptyset$

Événement certain

Contient toutes les issues.
Il est égal à $\Omega$.

Événement contraire

Noté $\bar{A}$, il contient tout ce qui n'est pas dans $A$.

Événements incompatibles

Deux événements qui ne peuvent pas se produire simultanément.

C Définition et propriétés

Loi de Laplace

$$P(A) = \frac{\text{Nombre de cas favorables}}{\text{Nombre de cas possibles}}$$

Règles fondamentales

  • Échelle $0 \le P(A) \le 1$
  • Totalité $P(\Omega) = 1$
  • Néant $P(\emptyset) = 0$

Calcul du contraire

$$P(\bar{A}) = 1 - P(A)$$

Utile pour gagner du temps

D Opérations sur les Événements

$\cap$

L'Intersection (ET)

L'événement noté $A \cap B$ est réalisé si les événements $A$ ET $B$ se réalisent simultanément.

A = "Obtenir un nombre pair"
B = "Obtenir un multiple de 3"
$A \cap B = \{6\}$
$\cup$

La Réunion (OU)

L'événement noté $A \cup B$ est réalisé si AU MOINS L'UN des deux événements se réalise (A, B, ou les deux).

A = $\{2, 4, 6\}$
B = $\{3, 6\}$
$A \cup B = \{2, 3, 4, 6\}$

Formule d'Addition

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

Cas particulier : Si $A$ et $B$ sont incompatibles (ils ne peuvent pas se produire en même temps, donc $A \cap B = \emptyset$), alors $P(A \cap B) = 0$.

La formule se simplifie en : $$P(A \cup B) = P(A) + P(B)$$
